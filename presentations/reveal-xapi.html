Check out the statements you're making

  • The slideshow experience is courtesy of Reveal.js, created by Hakim el Hattab
  • It is Open Sourced - built on HTML, CSS, and JavaScript
  • Popular with web developers
  • As many good open source software...
  • it does one thing really well...
  • ...it excels at delivering content navigation in browsers,
  • and browsers excel as ubiquitous content deliverers
  • so, as a standalone package will run in any compliant browser
  • Reveal is able to communicate with an xAPI endpoint because of a plugin, reveal.js-xapi-plugin
  • Reveal plugins augment capabilities, such as sending xAPI statements to an LRS endpoint
  • More plugins can be added or developed to augment capabilities, which modularly defines your content capabilities

Motivation for the plugin

  • Succint mechanism to send xAPI statements as a plugin
  • Demonstrate Open Sourc resources that can be easily 'patched' together
  • Advocation of using Open Source technologies for eLearning and Instructional Design
  • Experimentation
  • However, this format is not for every scenario
  • But it could be a fitting solution for the following learning scenarios
    • self-paced
    • MOOC
    • blended
    • interactive quiz and assessment

As a mechanism for collecting and reporting data,

  • Granular experience/statement reporting
  • Basis for Learning Analytics

Being based on web standards, it plays well with other standards based web technologies
